# 大作业设计题目：AI物联网专业智能助手APP

## 要求

实现一个移动端智能助手应用，至少包含以下功能模块。

## 1. 对话窗口

- 无限滚动对话窗口，支持异步数据响应。
- 会话持久化保存，以列表形式展示历史会话。
- 每个会话自动生成并保存摘要。
- 实现智能体会话的异步加载。
- 历史会话列表样式类似 DeepSeek 的会话列表。
- 支持多模态数据上传：拍照、文档、录音，支持会话中调用系统服务实现上述数据采集。
- 消息到达时 APP 内通知，并触发系统消息栏通知。
- 附加分：实现 Claw 模式本地智能体，可执行自动发送短信、自动拍照、自动下载服务。

## 2. 统计与监测页面

- 可视化展示对话条目统计数据。
- 显示智能体数量统计。
- 显示模型使用统计。
- 显示当前网络状态。

## 3. 设置与配置

- 可选择和切换大模型。
- 配置 APIKEY。
- APP 风格切换：暗色模式、浅色模式。

## 4. 动效

- 会话加载动效。
- Welcome Page 与首次加载引导动效。

## 评分标准

- 设计完成部分：70 分。
- 对话窗口：30 分。
- 智能体交互：20 分。
- 统计页面：15 分。
- 设置配置：15 分。
- 动效：15 分。
- 本地智能体：5 分。
- 答辩：15 分。
- 报告：15 分。
