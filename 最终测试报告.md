## 需要真机或真实 Key 验证的项目

- 真实 AI 回复：需要用户在设置页输入有效 API Key。
- 短信发送：需要有 SIM 卡并允许短信发送的真机。
- 拍照：需要设备存在系统相机 App。
- 通知弹出：Android 13 及以上需要用户授予通知权限。
- 下载服务：需要输入可访问的文件 URL。
- 图片理解：需要支持 vision 的模型，并在模型配置中开启“支持图片输入”。

## 结论

第 1 至第 18 阶段要求均已完成。项目可以正常构建，单元测试通过，未发现硬编码 API Key、TODO/FIXME 或阶段占位问题。剩余验证主要依赖真实 API Key、真机权限、支持 vision 的模型和现场网络环境。
